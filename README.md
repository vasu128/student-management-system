# 🎓 Student Management System (Python Tkinter)

A GUI-based desktop application built using Python and Tkinter to manage student records efficiently.  
The system supports full CRUD operations with CSV-based data storage and input validation.

## 🚀 Features

- ➕ Add new student records  
- ✏️ Update existing student details  
- ❌ Delete student records  
- 📋 View all students in a structured format  
- 💾 Data stored using CSV file  
- ✅ Input validation for accuracy  

## 🛠️ Tech Stack

- Python  
- Tkinter (GUI)  
- CSV (Data Storage)
